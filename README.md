# Java OOP Chess Game ♟️

A fully functional Chess game built from scratch using pure Java and Swing/AWT for the graphical interface. 

This project was developed to practice and demonstrate strong Object-Oriented Programming (OOP) principles, including inheritance, polymorphism, and encapsulation.

## 🚀 Technologies
* **Java** (Compatible with Java 8+)
* **Java Swing & AWT** (GUI and Event Handling)

## 🧠 Architecture Highlights
* **Object-Oriented Design:** The movement rules and logic are distributed across specific piece classes (`Bishop`, `King`, `Knight`, `Pawn`, `Queen`, `Rook`) inheriting from a generic `Piece` parent class.
* **Matrix Logic:** The board state is managed through a precise 2D array system, handling complex interactions like captures, valid moves, and collision detection.

## 🕹️ How to Run
1. Clone this repository.
2. Open the project in your preferred Java IDE (VS Code, IntelliJ, or Eclipse).
3. Ensure your JDK is set up correctly.
4. Run the `Main.java` file located in the `src/main` package.
